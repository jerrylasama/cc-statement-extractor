# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0-beta] - 2026-01-31

### Added
- Initial beta release.
- PDF to Markdown extraction using PaddleOCR.
- PII anonymization using Microsoft Presidio.
- Support for GLiNER and spaCy NLP engines.
- Custom deny list for specific redactions.
- Command-line interface with file-specific and force-processing options.
- Comprehensive documentation and automated tests.
